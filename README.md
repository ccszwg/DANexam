# DANexam

#### 项目介绍
danexam旨在提供一个简便易用、功能多样、囊括文理的\XeLaTex{}、\LuaLaTeX{}试卷模版包。

本包开发过程中有参考Philip Hirschhorn所作exam、鲍宏昌所作BHCexam、TJRSD所作GEEexam，在此特表感谢。

作者第一次做宏包开发，多有欠缺之处还望用户多加批评指正，尤其欢迎有兴趣者与我一起完善本包。
